Check if we are currently using impersonation

NURESTLoginController.equals(controller)[source]¶

Verify if the controller corresponds to the current one.

NURESTLoginController.get_authentication_header(user=None, api_key=None, password=None, certificate=None)[source]¶

Return authenication string to place in Authorization Header

If API Token is set, it’ll be used. Otherwise, the clear text password will be sent. Users of NURESTLoginController are responsible to clean the password property.

Returns:Returns the XREST Authentication string with API Key or user password encoded.
NURESTLoginController.impersonate(user, enterprise)[source]¶

Impersonate a user in a enterprise

Parameters:
  • user – the name of the user to impersonate
  • enterprise – the name of the enterprise where to use impersonation

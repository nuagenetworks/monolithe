classmethod BambouConfig.get_default_attribute_value(object_class, property_name, attr_type=<type 'str'>)[source]¶

Gets the default value of a given property for a given object.

These properties can be set in a config INI file looking like

[NUEntity]
default_behavior = THIS
speed = 1000

[NUOtherEntity]
attribute_name = a value

This will be used when creating a bambou.NURESTObject when no parameter or data is provided

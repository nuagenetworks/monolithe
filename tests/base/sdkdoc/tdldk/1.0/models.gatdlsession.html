class GATDLSession(username, enterprise, api_url, password=None, certificate=None)[source]¶

Returns the bambou2.NURESTLoginController of the current session

GATDLSession.reset()¶

Resets the session.

Resetting the session will flush the API stored API key. Any additional calls will require to call start, and a /me request will be reissued.

GATDLSession.start()¶

Starts the session.

Starting the session will actually get the API key of the current user

GATDLSession.stop_impersonate()¶

Stop impersonating a user
